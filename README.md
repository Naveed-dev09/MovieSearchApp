# 🎬 MovieSearch App

A modern and responsive movie discovery web application built with **React**, **Vite**, and **Tailwind CSS**. Browse trending and popular movies, search for your favorite titles, and enjoy a clean, responsive user interface.

## Features

*  Search for movies
*  Browse trending and popular movies
*  Fully responsive design
*  Fast performance with Vite
*  Modern UI built with Tailwind CSS
*  Movie cards with:

  * Poster
  * Title
  * Rating
  * Release Date
  * Overview
* 🎠 Interactive movie sliders
* ✨ Smooth animations using Framer Motion

##  Tech Stack

* React
* Vite
* Tailwind CSS
* JavaScript (ES6+)
* TMDB API
* Framer Motion
* Swiper.js

##  Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/movie-search-app.git
```

### Navigate to the project

```bash
cd movie-search-app
```

### Install dependencies

```bash
npm install
```

### Create a `.env` file

```env
VITE_TMDB_API_KEY=your_tmdb_api_key
```

### Start the development server

```bash
npm run dev
```
